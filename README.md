# 🌐 Portfolio

My personal corner of the web. A simple portfolio site with my skills, projects, and a downloadable CV.

🔗 **[View Live](https://mr-heain.github.io/portfolio)**

---

## 📄 Sections

- **About** — who I am
- **Skills** — what I'm working with
- **Projects** — things I've built
- **Contact** — how to reach me
- **CV** — view or download my resume
